# 短视频爆款文案执行

## 核心底层逻辑：买方思维

用户留下来看的唯一理由是"这对我有价值"。

### 买方思维3问（动笔前必须回答，答不上来不许写）
1. 这个用户今天刷视频是为了什么？
2. 这条视频能给他带走什么？
3. 他看完之后，生活有什么变化？

## 6大用户价值维度（必须选定主攻方向）

| 价值类型 | 用户内心OS | 适用场景 |
|---------|-----------|---------|
| 情绪价值 | "看完心情好了/爽了/感动了" | 种草、品牌故事 |
| 自身价值 | "让我显得更好/更健康" | 美妆、穿搭、食品 |
| 职场价值 | "让我工作更厉害" | 技能、工具 |
| 收入价值 | "让我省钱/赚钱" | 带货、选品 |
| 学习价值 | "让我长见识了" | 知识、测评 |
| 社交价值 | "我要转发/显得懂行" | 热点、反常识 |

同一产品不同维度写法完全不同，必须先选定再动笔。

## 爆款结构公式（视频骨架）

| 时段 | 作用 | 买方视角 | 失败后果 |
|------|------|---------|---------|
| 前3秒（钩子） | 让人停下来 | "这和我有关吗？" | 直接划走 |
| 3-15秒（共鸣） | 让人感受到价值 | "他懂我，继续看" | 觉得没关系划走 |
| 15-40秒（方案） | 给用户带走点什么 | "这对我有用吗？" | 没收获划走 |
| 40-55秒（证明） | 让人信 | "是真的吗？" | 觉得是广告划走 |
| 最后5秒（行动） | 让人动 | "我做什么？" | 互动率低不推流 |

核心原则：
- ❌ 产品视角："我们的龙虾水质好个头大"
- ✅ 用户价值视角："为什么同样是龙虾，有人吃完惊艳有人觉得普通？"

## 5种标题公式

| 类型 | 公式 | 带货用 |
|------|------|-------|
| 数字型 | N个方法让你XXX | ✅ |
| 反常识型 | 你以为X，其实Y | |
| 对比型 | 同样是X，为什么A赢B输 | |
| 恐惧型 | 再不X就晚了 | ✅ |
| 好奇型 | X之后发生了什么 | |

## 流量钩子

完播悬念（每10秒埋1个）：
- "后面发生的事我完全没想到…"
- "重点来了，这才是最关键的…"

评论区钩子（结尾必须有）：
- 争议型："你们觉得贵吗？"
- 求助型："你们当地多少钱？"

## 写作金标准
1. 前3秒用建模表确认的钩子句
2. 共鸣段必须站用户角度，体现价值维度
3. 方案段说"用户能得到什么"，不是"产品有什么"
4. 每10秒一个完播悬念点
5. 结尾必须有行动指令+评论区钩子
6. 合规：无违规词、无功效词
7. 口语化：不许书面语

## 执行产出物清单（缺一不可）
1. 标题（最终版）
2. 封面文字
3. 逐秒口播文案
4. 字幕文案
5. 评论区置顶文案
6. 发布正文描述（含话题标签）
